  Плагин для построения маршрутов.

## Требования

- **QGIS** версии 3.x
- **Python** 3 (встроенный в QGIS или внешний)
- Необходимые библиотеки: `os`, `qgis.core`, `qgis.utils` (стандартные для среды QGIS).

## Установка плагинов

Вы можете установить плагины двумя способами:

### Способ 1: Установка из ZIP через QGIS

1.  Откройте QGIS.
2.  Перейдите в меню **Плагины (Plugins) → Управление и установка модулей (Manage and Install Plugins)**.
3.  Выберите вкладку **Установить из ZIP (Install from ZIP)**.
4.  Укажите путь к файлу `RouterPlugin.zip` или `ModelParkingPlugin.zip`.
5.  Нажмите **Установить (Install Plugin)**.

### Способ 2: Ручная установка

1.  Распакуйте архивы `RouterPlugin.zip` и `ModelParkingPlugin.zip`.
2.  Скопируйте папки с плагинами в директорию профиля QGIS:
    - **Windows:** `C:\Users\%USERNAME%\AppData\Roaming\QGIS\QGIS3\profiles\default\python\plugins\`
    - **Linux:** `~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/`
3.  Перезапустите QGIS и активируйте плагины в менеджере модулей.

## Использование

### RouterPlugin

1.  Откройте панель плагина через меню или иконку на панели инструментов.
2.  Задайте входные слои:
    - Слой остановок/точек старта.
    - Слой графа УДС (линии).
    - (Опционально) DEM для учета рельефа.
3.  Настройте параметры (скорость, предельное время/расстояние).
4.  Запустите расчет. Результатом будет слой полигонов (изохрон) или линий маршрутов.

### ModelParkingPlugin

1.  Загрузите слои парковок и объектов притяжения.
2.  Укажите параметры моделирования (радиус доступности, коэффициенты спроса).
3.  Плагин рассчитает показатели обеспеченности или дефицита мест.

### Запуск скриптов

Скрипты `5-test-1.py` и `1-reborn-3.py` предназначены для запуска через **Python Console** в QGIS:

1.  Откройте консоль (Ctrl+Alt+P).
2.  Откройте файл скрипта в редакторе консоли.
3.  Нажмите кнопку "Run Script".
